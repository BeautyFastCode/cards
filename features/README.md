# Behat

## Gherkin Syntax

```Gherkin
Feature: {custom_title}
  In order to {A}
  As a {B}
  I need to {C}
```
- {A} = the benefit or value of the feature
- {B} = the role (or person) who will benefit
- {C} = short feature description
